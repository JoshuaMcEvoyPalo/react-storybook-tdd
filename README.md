# React Storybook TDD Repo

This is a repo to demonstrate the importance of TDD on frontend React projects

## Setting up repo

Clone:
```bash
git clone https://github.com/JoshuaMcEvoyPalo/react-storybook-tdd.git
```

Running project:
```bash
npm run start
```

Running Storybook:
```bash
npm run storybook
```
